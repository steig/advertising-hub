# windsurf Integration

Integration instructions for using Advertising Hub agents with windsurf.

## Install

```bash
./scripts/install.sh --tool windsurf
```

## Usage

Activate agents by referencing them in your windsurf sessions:

```
Use the PPC Campaign Strategist agent to help with this Google Ads account.
```

See the main [Integrations README](../README.md) for full details.
