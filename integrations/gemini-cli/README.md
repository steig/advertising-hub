# gemini-cli Integration

Integration instructions for using Advertising Hub agents with gemini-cli.

## Install

```bash
./scripts/install.sh --tool gemini-cli
```

## Usage

Activate agents by referencing them in your gemini-cli sessions:

```
Use the PPC Campaign Strategist agent to help with this Google Ads account.
```

See the main [Integrations README](../README.md) for full details.
