# claude-code Integration

Integration instructions for using Advertising Hub agents with claude-code.

## Install

```bash
./scripts/install.sh --tool claude-code
```

## Usage

Activate agents by referencing them in your claude-code sessions:

```
Use the PPC Campaign Strategist agent to help with this Google Ads account.
```

See the main [Integrations README](../README.md) for full details.
