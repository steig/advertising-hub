# Catalog Ads (Dynamic Product Ads)

## Setup Sequence

1. Create Product Catalog in Commerce Manager
2. Upload product feed (scheduled feed URL or manual)
3. Install Meta Pixel with ViewContent, AddToCart, Purchase events
4. Create Campaign → Advantage+ catalog ads

## Feed Requirements

Required fields: `id`, `title`, `description`, `availability`, `condition`, `price`, `link`, `image_link`, `brand`

## Pro Tips

- Product Set segmentation is your main optimization lever
- Use "broad audience" for prospecting, "retargeting" for warm audiences
- Creative overlays (price, discount badge) significantly improve CTR
- Feed quality directly impacts delivery — fix disapproved products first
