# Advantage+ Campaigns

## Advantage+ Shopping

Fully automated shopping campaign type. You provide the catalog, budget, and country — Meta handles everything else.

### When to Use
- Existing catalog with 50+ products
- Sufficient conversion history (50+ purchases/week)
- Want to maximize ROAS with minimal management

### When NOT to Use
- New accounts with no conversion data
- Need granular audience or placement control
- Budget is too small for the algorithm to learn

## Advantage+ App Campaigns

Similar automation for app install campaigns. Requires the SDK integrated for event optimization.

## Key Settings

- **Existing customer budget cap**: Set this to prevent spending 90% of budget on existing customers
- **Country targeting**: Only required setting — everything else is automated
- **Creative**: Provide diverse creative assets — the algorithm needs variety to test
