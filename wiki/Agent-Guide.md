# Agent Guide

## What Are Agents?

Agents are markdown files that give AI tools (Claude, Cursor, Gemini) a specialized personality, expertise, and workflow for a specific domain. They're not generic prompts — they're deep specializations with defined capabilities, tooling references, and success metrics.

## Using Agents

### Quick Activation

```
# In Claude Code, Cursor, or similar:
"Use the PPC Campaign Strategist agent to audit this Google Ads account structure"
"Use the Budget Allocator agent to recommend how to split $100K across Google and Meta"
"Use Buddy to figure out which agent I need for this Pinterest shopping question"
```
