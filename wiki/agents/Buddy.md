# Buddy — The Orchestrator

Buddy is the meta-agent that routes your questions to the right specialist. When you're not sure which agent you need, start with Buddy.

## How Buddy Works

1. You describe what you need
2. Buddy identifies the platforms and domains involved
3. Buddy activates the appropriate specialist(s)
4. If multiple agents are needed, Buddy coordinates the workflow

## Example Routing

| You Ask | Buddy Routes To |
|---------|----------------|
| "Audit my Google Ads" | Paid Media Auditor + google-ads-mcp |
| "Where should I spend $50K?" | Budget Allocator |
| "Set up LinkedIn CAPI" | LinkedIn B2B Strategist |
| "Why don't conversions match?" | Attribution Analyst |
| "Build a Meta Ads MCP" | → MCP Development Guide |
