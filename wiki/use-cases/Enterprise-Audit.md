# Use Case: Enterprise Audit

Full advertising audit across all active platforms.

## Process

1. **Inventory**: List all active platforms, accounts, and budgets
2. **Per-Platform Audit**: Deploy platform-specific agents for each
3. **Cross-Platform Analysis**: Use cross-platform agents to identify gaps
4. **Unified Report**: Consolidated findings with prioritized recommendations

## Agents Used

- Paid Media Auditor (per-platform)
- Attribution Analyst (cross-platform measurement)
- Budget Allocator (spend optimization)
- Competitive Intel (market positioning)
- Reporting Unifier (consolidated dashboard)
