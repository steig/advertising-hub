# Use Case: Cross-Platform Campaign Launch

Launching a campaign across Google Ads, Meta, LinkedIn, and Pinterest simultaneously.

## Agent Workflow

1. **💸 Budget Allocator** — Determine optimal budget split across platforms
2. **👥 Audience Architect** — Build consistent audience segments for all platforms
3. **💰 PPC Campaign Strategist** — Google Ads campaign architecture
4. **📱 Paid Social Strategist** — Meta and LinkedIn campaign design
5. **📌 Pinterest Visual Commerce** — Pinterest shopping campaign
6. **📡 Tracking Specialist** — Unified conversion tracking across all platforms
7. **📈 Reporting Unifier** — Cross-platform reporting dashboard

## Key Decisions

- **Budget split**: Use Budget Allocator's diminishing returns analysis
- **Audience consistency**: Same customer lists uploaded to all platforms
- **Tracking**: Unified UTM structure + server-side tracking where available
- **Attribution**: Set consistent attribution windows before launch
