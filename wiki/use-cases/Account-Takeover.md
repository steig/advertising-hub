# Use Case: Account Takeover

Systematic approach to taking over a new advertising account using Advertising Hub agents.

## Agent Sequence

1. **📋 Paid Media Auditor** — Comprehensive assessment across 200+ checkpoints
2. **📡 Tracking Specialist** — Verify conversion tracking accuracy BEFORE optimizing
3. **💰 PPC Campaign Strategist** — Redesign account architecture based on audit findings
4. **🔍 Search Query Analyst** — Clean up wasted spend from search terms
5. **✍️ Creative Strategist** — Refresh ad copy and extensions
6. **📈 Reporting Unifier** — Build unified reporting dashboard

## With MCP (Google Ads)

```
1. Activate Paid Media Auditor agent
2. Agent uses google-ads-mcp to pull live account data
3. Agent runs audit against 200+ checkpoints using real data
4. Produces prioritized findings with specific fixes
5. Hand off to PPC Strategist for restructuring
```

## Timeline

- **Week 1**: Audit + tracking verification
- **Week 2**: Account restructure + search term cleanup
- **Week 3**: Creative refresh + reporting setup
- **Week 4**: Optimization begins with clean foundation

## Result

Tracking verified, waste eliminated, structure optimized, creative refreshed — all within the first 30 days.
